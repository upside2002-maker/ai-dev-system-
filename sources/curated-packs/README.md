# Curated Packs

Здесь лежат не сырые ссылки, а уже переработанные наборы качественных паттернов.

Идея простая:

- интернет дает примеры;
- мы делаем отбор;
- потом превращаем отбор в локальную систему знаний;
- уже этим кормим агентов.

## Доступные наборы

- `HASKELL_TYPED_CORE_TOP20.md`
- `PYTHON_SERVICES_TOP20.md`
- `REACT_OPERATOR_UI_TOP20.md`

## Как применять

Перед конкретной задачей агенту дается:

1. overlay проекта
2. playbook слоя
3. relevant top-20 pack
4. 1-2 локальных snippets из самого проекта

Это дает гораздо лучший результат, чем просто "посмотри GitHub".
