# Пирамидинг серебра через SLV + защитный пут — извлечённая стратегия

**Источник:** [СЕРЕБРО: Как заработать х10 на пирамидинге? — Dmitry Solodin](https://www.youtube.com/watch?v=nswcGlSh4Gg), 2:10, 11.07.2025.
Ядро стратегии: [00:58:16–01:13:38](https://www.youtube.com/watch?v=nswcGlSh4Gg&t=3496s). Позиция у автора реально открыта (вход 32.4), лестница дальше — «планирую, пока в теории» [01:08:33].

## Главный фрейм

Двухслойный тезис: **структурный дефицит серебра** (≈80% предложения — попутная добыча → неэластично; дефицит-2023 = 184 млн унций; солнечная энергетика к 2027 потребует >20% годовой добычи) **плюс монетарная девальвация доллара** → длинный тренд с целью ~$100 (линейная шкала) / ~$500 (лог-шкала, горизонт 2050+). Под тезис строится опционная конструкция с асимметрией: риск каждого цикла ~2.5%, доход не ограничен, прибыль реинвестируется с сохранением первоначального риска.

## Механика по шагам

1. **База (синтетический колл):** акции SLV (вход 32.4) + ITM-пут страйк 35 дальней серии (2026/2027). Риск конструкции ~2–2.5% при любом падении. Без управления до $47: доход ~30–35%, ~x12 к риску [00:58:16–01:00:26].
2. **Рейз на 35:** цена дошла до страйка → ролл пута 35→40 (ATM-пут продаётся с максимальной временной стоимостью, ITM-40 покупается); чистая фиксация ≈ +1.2 $/акц; позиция удваивается по 35; итог 2×SLV + 2×пут-40 [01:01:28–01:05:30].
3. **Хедж-пропорция:** 10 путов на 1000 акций (полное покрытие); опционально 11 — over-hedge, профиль на сильном падении уходит в плюс [01:04:29].
4. **Лестница:** 32.4 → 35 → 40 → 45 → точка прорыва 50; на каждом уровне ролл страховки + наращивание [01:07:33]. Кратность x2 заявлена только для первого рейза.
5. **Банкинг:** на каждом рейзе часть прибыли — «в банк»; следующий рейз не на всю прибыль. Доли не формализованы («копеечку, 2, 5») [01:12:38].
6. **Фаза 2 — OTM-коллы:** при пробое $50 (SLV) / $47 спот (активация «чашки с ручкой») вся позиция меняется на OTM-коллы (пример: цена 50 → колл 70 за ~$0.80; плечо до 100–1000x). Накопленное к этому моменту ≈ x30 от первоначального риска; теоретический потолок цепочки x300, «так агрессивно никто не работает» [01:08:33–01:12:38].

## Что осталось «на глаз» (10 пунктов из извлечения)

Доли банкинга; кратность рейзов после первого; триггер уровня (касание/закрытие); размер over-hedge; выбор/ролл серий; все параметры OTM-фазы; условие отмены («спокойно зафиксирую» без уровня); вероятность 90%; состав 10 акций; размер позиции.

## Оговорки по качеству данных

Годы целей искажены капшенами («4кового года» ≈ 2040; «к 2008» ≈ 2030-е — низкая достоверность). Противоречие в самом видео: попутная добыча «80%» vs «2/3». Лестница 32.4/35/40/45/50 — цены SLV, а сигнальные $47–48 — спот серебра: две разные шкалы, автор использует обе. «Бессрочные опционы» (вопрос зрителя) автор разбирает как маркетинг: это роллируемые годовые опционы с тем же временным распадом [01:44:18–01:46:18].
