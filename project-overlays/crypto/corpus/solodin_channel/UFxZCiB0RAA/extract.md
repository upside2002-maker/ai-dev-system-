## 3. Фундаментал / механика рынков (обучающие блоки)

- **Тариф как изъятие ликвидности**: тариф-доход бюджета = либо вычет из прибыли компаний, либо инфляция для потребителя — «бесплатных» денег у бюджета не бывает [00:31:40].
- **Дефицит = скрытая эмиссия**: дефицит увеличивает долг, что эквивалентно скрытой денежной эмиссии и обесцениванию валюты [00:15:20].
- **Рефинансирование госдолга**: снижение ставки удешевляет обслуживание долга только через рефинанс (новый долг по новой ставке гасит старый) [00:14:17].
- **Стагфляция vs дефляция и реакция ЦБ**: при стагфляции ЦБ не может резко снижать ставку (инфляция «шурует»); при дефляции — руки развязаны, ставку в ноль [00:45:05]–[00:46:06].
- **Опережающие индикаторы**: арендные ставки для новых съёмщиков → CPI-rent [00:33:43]; Global M2 (сдвиг +3 мес) → фондовый рынок [01:30:02].
- **Накопление/распределение (Вайкофф-логика)**: объёмы накопления видны внизу в абсолютной шкале (логарифм маскирует); распределение наверху тянет цену выше, т.к. продавцу выгодна высокая цена [01:22:54]–[01:25:56].
- **Байбек и EPS**: крупный байбек (>3% капы) при большом FCF механически поднимает EPS и поддерживает цену; FCF — гарантия устойчивости выкупа [00:51:11], [01:36:12].
- **Мультипликаторы P/E, P/S (TTM)**: оценивают по прибыли и выручке; падение мультипликатора при растущих прибыли/выручке = удешевление компании. TTM-сглаживание по 12 мес может искажать среднюю доходность [00:54:16]–[00:55:16].
- **Маржинальность как потолок**: операционная доходность имеет «потолок» (для NICE ~30% при валовой 67%; для DOCN до ~20%) — выше расти трудно [00:55:16], [01:45:24].
- **Волновой анализ (Эллиотт)**: типы — импульс, зигзаг/двойной зигзаг (W-X-Y), плоскости (regular/expanded EFL, running RFL), диагонали (расширяющаяся), треугольники (A-B-C-D-E, волны сужаются, E не выходит за C); зигзаги часто идут в канале по совпадающим пикам [00:21:27]–[00:25:32], [00:48:07]–[00:50:10], [01:33:10]–[01:35:12].
- **Жильё ~30% ВВП США, малый бизнес ~55% ВВП** — масштаб затронутых секторов [00:40:57], [00:46:06].
- **Жильё как «упаковка денег»**: квартира — не трата, а форма хранения капитала (можно продать и жить на эти деньги); дорогой ремонт — трата (не окупается при продаже) [01:12:38]–[01:13:39].

## 4. Риск-менеджмент с цифрами

- В фазе «много сигналов на коррекцию» — **не вставать в полный рост**, готовность пропустить первый удар [00:28:36].
- **Хедж рисковых позиций в моменте**: опасные бумаги покупаются сразу со страховкой; безопасные (отъехали 50–60% от хая) — без [00:29:38].
- **«Бери ношу по себе, чтобы не падать при ходьбе»** — не наращивать расходы под пиковый доход; не садиться на большое плечо: тренд дохода не бесконечен (как откаты −20/−30/−40/−90% на рынке) [01:15:41]–[01:16:43].
- **Подушка/заначка**: держать запас на «недоступные» события (болезнь, заморозка активов, инфляция) [01:11:38]–[01:12:38].
- **Кассовый разрыв**: даже при больших активах можно остаться без ликвидности «в моменте» — следствие большого плеча [01:15:41].
- По DOCN: спокойное усреднение в зоне накопления вплоть до ~$22 как заранее заложенный подвариант, без паники [01:41:21].
