## 1. Стратегии и торговые механики

### 1.1. Хеджированное «поджатие» (накопление крупной позиции под защитой опционов)
- Суть: держится крупная позиция в качественной бумаге; постоянно покупается защита (опционы пут/страховка). При резком падении прибыль от страховки фиксируется, позиция «раскрывается» по более низкой цене → средняя падает. Когда бумага «распрямляется», поджатая позиция даёт большую прибыль. [01:43:36]–[01:45:38]
- Кейс PayPal (PYPL): крупнейший холдинг автора, ~8–9% счёта [01:43:36], [01:46:39]. Покупал с 2020/2022, средняя ~80; за счёт перероллов страховки средняя резко снижена, позиция в небольшом плюсе. [01:44:36]
- Зачем страховать: без страховки бумага «может уйти на 35», экспозиция вниз неприемлема при таком размере → опционы снижают бета-риск, реальная просадка меньше номинальных 9%. [01:43:36], [01:46:39]
- Механика ролла: «постоянно заканчивается страховка — открываю новую»; на падении страховку «перероллировал», на отскоке вышел в плюс. [01:44:36]
- Название стратегии у автора: «поджатие», разбирается на его курсе «Прокачка». [01:45:38]
- Формализуема программно: частично (логика ролла пут-хеджа по триггерам цены формализуема; выбор момента и «качество бумаги» — дискреционны).

### 1.2. Пирамидинг через опционы (наращивание без роста линейного риска)
- Суть: увеличение позиции по тренду делается не докупкой акций, а опционными конструкциями — «увеличиваю позицию, но снимаю риск». [01:48:42]
- Кейс LYFT: покупал по ~14, продолжает наращивать через опционы (не линейно). Прямо предупреждает: не воспринимать как «просто докупать акции». [01:48:42]
- Формализуема: частично (правила добавления по пробою/тренду формализуемы; конкретная опционная конструкция в видео не раскрыта детально).

### 1.3. Покупка «связкой/парой» однотемных лидеров
- Суть: когда неизвестно, кто из двух лидеров ниши выиграет, берутся оба пакетом. [00:56:20], [00:57:21]
- Кейсы: LLY + NVO покупать вместе (оба — препараты для похудения/диабета, оба строят производство в США); фарму автор «взял просто пакетом», т.к. не знает, какая бумага вырастет, индекс брать не хочет — берёт крупные, но непохожие бумаги. [00:18:24], [00:56:20], [00:57:21]
- Формализуема: да (корзина из N лидеров сектора с равными весами).

### 1.4. Вход по «пробою → ретесту → продолжению» (классическая ТА-механика)
- Суть: бумага пробивает сопротивление, тестирует линию пробоя сверху, затем продолжает рост. Покупка на ретесте линии пробоя. [00:43:00], [00:47:05]
- Кейс PFE: куплен по ~23 «за день до взлёта», на повторном тесте пробитого сопротивления (недельный ТФ). [00:43:00]
- Оговорка автора: ретест не всегда случается — «двойной тест мог уже быть тестом», бумага может убежать без отката; тогда тест будет выше. [00:47:05]
- Формализуема: да (breakout + retest + entry).

### 1.5. Вход после треугольника / на выходе из многолетнего боковика
- Суть: после треугольника обычно выход по направлению предыдущего движения; особо ценен выход из очень долгого боковика. [00:18:24] (AMGN), [00:24:28] (GSK).
- Кейс GSK: выход из 25-летнего треугольника/боковика (с 1998 г.), пять волн зарегистрированы, forward P/E ~8; «технически идеально». [00:24:28], [00:25:28]
- Формализуема: частично (детект треугольника/боковика + breakout формализуем; волновая разметка — дискреционна).

### 1.6. Ожидание пробоя ключевого уровня перед действием (для нерисковых)
- Универсальное правило автора по многим тикерам: «не рискуй — дождись выхода/пробоя уровня». Раньше = больше риск, позже = меньше риск; выбор за инвестором. [01:26:04], [01:27:05], [01:37:30].

## 3. Фундаментал и механика рынков (обучающие takeaway)

- 100% тариф на фарму с 1 октября — только если компания НЕ строит завод в США; исключение для уже начавших стройку (in-building). Логика Трампа: загнать бигфарму строить заводы в США. [00:02:48], [00:03:49]
- TrumpRX — платформа прямой продажи рецептурных лекарств потребителю без страховки/посредников. Фарма теряет немного: убираются откаты/комиссии дистрибьюторам → можно снижать цены без удара по марже. [00:05:54]
- Механизм «индульгенции»: инвестируй в производство в США → получаешь тарифную передышку. Pfizer: скидки до 85% незастрахованным + продажи Medicaid дешевле → передышка до 3 лет; на объявлении +6–8% в день, весь сектор +4–5%. [00:06:55], [00:07:55], [00:08:56]
- Почему фарма не платит большие дивиденды/байбэки: большие capex и инвестиции в R&D (пайплайн) → мало свободного денежного потока. Стабильные, но небольшие дивиденды. [00:33:43], [00:34:46]
- Где искать сильные дивиденды: финтехи/технологические (быстрый рост, низкие расходы, высокая маржа → большой FCF) и особенно REIT (по закону обязаны распределять бóльшую часть выручки в дивиденды; по сути финансовый бизнес). [00:34:46], [00:35:46]
- Экономика пайплайна биотеха: ранние стадии дёшевы (млн $/год), 3-я фаза = широкое тестирование = сотни млн $ косты → именно на 3-й фазе сжигается кэш (кейс MRNA). [01:15:46], [01:17:53]
- Капексы ИИ: «поставщики лопат» (NVIDIA) деньги уже заработали и поставили железо; «круговорот» денег между Oracle/NVIDIA/IBM и пр. Замедление капексов → падение темпов выручки → переоценка. [01:07:30], [01:08:33]
- Волновая теория (ликбез): импульсы и диагонали = motive waves, двигают цену из A в B; коррекция не может содержать 8 импульсных подволн (8 «иксов» = импульс). Коррекция ≤ ~150–200%, не 8x. [01:35:26], [01:36:27]
