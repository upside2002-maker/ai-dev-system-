# Торговля опционами на небольшом счёте — извлечённые стратегии и правила

**Источник:** [Торговля Опционами на небольшом счёте — Dmitry Solodin](https://www.youtube.com/watch?v=nHv14MWVymE), 2:45:53, 31.05.2024.
Автор — опционный трейдер с ~20-летним стажем (с 2004), публичная статистика: профит-фактор ~2.3 при винрейте ~60% [02:00:51]. Торгует в основном **покупки опционов** на американском рынке (опционы на микрофьючерс S&P 500). Извлечено из автотранскрипта; таймкоды кликабельны по главам в описании видео.

---

## Главный фрейм видео

**Маленький счёт ($2 000–10 000) = только ПОКУПКА опционов. Никаких продаж.** Розница покупает опционы, профессионалы продают [00:53:55]. Любая стратегия с проданной непокрытой ногой тянет за собой маржу, риск досрочного assignment (американский стиль — поставку могут истребовать в любой день) и мгновенный маржин-колл: счёт $2 000, проданный колл на Tesla по $170 → обязанность поставить акций на $17 000 [00:30:14].

Вторая ось: **специализация вместо диверсификации** [01:31:16]. На малом счёте нет денег на 5–7 тикеров — 1–2 ликвидных инструмента, минимальный набор стратегий.

Реалистичная цель: **~20% годовых** при профессиональной работе. Погоня за «иксами» через повышение риска = почти гарантированный слив; истории успеха — ошибка выжившего [01:53:43].

## Что РАБОТАЕТ на малом счёте

### 1. Long call / long put (база)
Риск ограничен премией — естественный стоп, физический стоп не нужен [02:00:51–02:02:52]. Long call ≈ лонг со стопом, long put ≈ шорт со стопом.

### 2. Вертикальный дебетовый спред (bull call / bear put)
- Купил колл за $300, продал дальний за $200 → затраты и максимальный риск $100 [02:02:52].
- Живой пример: Tesla long call 182.5 / short call 202.5 [02:03:53].
- Ширина страйков — регулятор: сдвигаешь ближе → меньше риск и прибыль, раздвигаешь → больше оба [02:06:59].
- Стопы не нужны, позиция не требует присмотра при движении против; внимание — только на фиксацию прибыли [02:04:56].

### 3. Long strangle / straddle (покупка волатильности)
- Покупка пута + колла одной экспирации; ratio-перекос (2 пута + 1 колл = медвежий уклон) [02:07:59].
- ГО почти не требуется — подходит малым счетам.
- **Условие входа: низкая IV с ожиданием роста + ожидание сильного движения без уверенности в направлении.** Главная ошибка применяющих — неумение работать с волатильностью [02:07:59–02:09:01].

### 4. «Пирамидинг» — главная стратегия для малых счетов [02:09:20–02:24:35]
Авторская стратегия, позиционируется как созданная для малых счетов.

**Механика (учебный пример):**
1. БА $20 → купить OTM-колл страйк 30 за $1.20 → риск $120 (1 контракт).
2. БА дошёл до $40 → опцион ~$10.20 (~10x) → продать (~$1 020).
3. Реинвест: 7 контрактов страйк 50 по $1.50 = $1 050. Зафиксировано ~$900 прибыли, новый риск снова ≈ первоначальный.
4. БА $70 → ~$18.5 × 700 = ~$13 000.

**Принципы:**
- Риск ни на одной итерации не превышает первоначальный (~$120) — вся прибыль реинвестируется.
- Прогрессия контрактов 1 → 3 → 9 → 25, потенциал до x75 с одной «линии» [02:23:33].
- На счёте $10 000 — до **3 параллельных линий**; часть линий сгорит, это нормально [02:17:13].
- Вход — по волновой разметке Эллиотта (после коррекции 2-й волны в расчёте на 3-ю) [02:10:03].
- Критерий актива: нужен большой потенциальный диапазон (с $2 до $65 — да; с $500 до $700 — пирамида не строится) [02:24:35].

**Живой пример — ESPR** [02:18:35–02:23:33]: цена ~$2, исторический максимум $122, разметка ABC-коррекции; декабрьская серия неликвидна → взята сентябрьская, страйк 2.5, премия $0.58 × 2 контракта = риск $116. Рост БА до $4.73 → опцион $2.80 → позиция $560, профит $444.

### 5. Система «трёх ячеек риска» [01:48:38–01:50:39]
Замена недоступной диверсификации: счёт $10 000 → 3 ячейки по $100 риска в 3 **разных** активах одновременно. Ячейка закрылась (профит или убыток) → открываешь новую. Суммарный одновременный риск ≤ $300.

## Что НЕ работает на малом счёте (и почему)

| Стратегия | Почему нет | Таймкод |
|---|---|---|
| Продажа опционов (любая непокрытая) | залог ≈ стоимость БА, ограниченная прибыль / неограниченный риск, неизбежная поставка | [01:13:45] |
| Кредитные спреды | нужна диверсификация 5–7 тикеров + резерв под поставку | [01:00:09] |
| Covered call | базовый актив съедает весь счёт; это стратегия для инвесторов с готовым активом | [00:55:10] |
| Protective put | = синтетический колл; дешевле просто купить колл | [02:25:40] |
| Синтетика (long call + short put) | ГО как за акции | [02:36:20] |
| Бэк-спреды (ratio back spread) | непокрытая нога → маржа + assignment-риск | [00:25:08] |
| 0DTE (ежедневные опционы) | PDT-лимит 3 дейтрейда/5 дней на счетах <$25k; 82% ритейла в 0DTE теряют | [00:51:50] |
| Кондоры/бабочки | 4 ноги × вход-выход = 8 комиссий, съедают плоский профит $10–20 | [01:33:17] |
| Покупка опционов под квартальный отчёт | IV crush: волатильность падает после отчёта, минус даже при верном направлении | [02:38:20] |

## Риск-менеджмент (правила с цифрами)

- **1–2% счёта на сделку** — норма; до 5% — только с проверенной стратегией; меньше ~0.5% — позиции «не помещаются» [01:46:37, 01:55:46–01:57:47].
- Счёт $10 000 → риск $100–200/сделку → 10 убытков подряд = −10–20% депо, приемлемо [01:47:38].
- Математика просадки: −70% требует +230% для возврата [01:48:38].
- **Мат. ожидание: целиться в риск/прибыль ~1:3** [01:58:25].
- **Физические стопы на опционах — никогда**: неликвид, проскальзывание, неисполнение. Вместо них алерты по базовому активу + ручной выход [01:05:45]. Ловушка: стоп на 4-ногую конструкцию = 4 дейтрейда разом = нарушение PDT [01:06:33].
- **Маржинальная загрузка ≤ 50%** от ликвидационной стоимости счёта (у автора <10%) [01:18:50]. IB режет маржин-колл автоматически и «злобно» — начиная с прибыльных ликвидных позиций [01:19:51].
- **Комиссии ≤ 30% от ожидаемого гейна стратегии**, иначе стратегия не подходит [01:34:18]. Пример: риск/доход 3:1 ($10/$30) после комиссий превращается в ~1:1 [01:35:22].
- Optimal f Винса (математический оптимум до 67% на сделку) — на практике неприменим; оценивать риск слива через Монте-Карло [01:54:44–01:56:47].

## Практические эвристики и параметры

- **ATM-премия ≈ 10% цены базового актива** (нормальная серия); ITM — до ~20% [01:27:04, 01:10:39].
- Под счёт $5 000 искать акции **$10–30** (премия ~$300/контракт); фильтр скринера $5–40 [01:29:12].
- **Выбор экспирации: ожидаемое время движения + запас** (движение за 2 недели → 4-недельный опцион; за 2 месяца → 3-месячный). Впритык — гамма-риск [02:26:30].
- **Скорость тренда ~2 SD, >3 SD практически не бывает** — фильтр реалистичности цели и срока [02:29:37]. Не ждать рост круче предыдущего падения [02:28:37].
- **Ликвидность**: страйки ATM самые расторгованные; круглые страйки ликвиднее; OI «расторгованного» страйка ~10 000+; нулевой OI — не входить; узкий bid-ask (1.03/1.05) = ликвид, редкие страйки (5/10/15) = неликвид [01:39:45–01:46:50]. Регулярные серии ликвиднее промежуточных [02:30:39].
- **Deep ITM** — стоимость владения близка к безрисковой ставке, «плечо почти на халяву»; ближняя 7-дневная серия ≈ 17% годовых временной стоимости [02:32:39].
- **PDT-правило**: счёт <$25 000 → максимум 3 внутридневных сделки за 5 рабочих дней; нарушение — блок на 90 дней [01:02:50].
- Минимальные суммы: США реально от $10 000 (комфортно $30 000+); РФ от 50–100 тыс. руб [00:19:40].
- Комиссии: ~$1/контракт; минимальная плата за неактивность ~$50/мес на малых счетах; маркет-дата $7–10/мес (россиянам сейчас не продают) [01:32:35–01:38:26].

## Инструменты

- **Interactive Brokers / TWS** — основной: бесплатный просмотр IV, столбец Time Value (стоимость владения в % годовых), Initial/Maintenance Margin [00:05:30, 02:31:39].
- **Подборщик стратегий OptionStrat** (или аналог): тикер + направление + target + лимит риска → готовые конструкции. Оценка автора: «база для новичков» [02:34:30].
- Бесплатный опционный калькулятор на сайте МосБиржи [00:41:35].
- РФ-терминалы: SmartX (ITI Invest), БКС; Тинькофф — опционы только на покупку, спред не построить [00:44:43, 01:23:01].
- ТА: волны Эллиотта, Фибоначчи, каналы стандартных отклонений.

## РФ-специфика (оценки автора «со стороны», он не резидент)

- После 2022 ликвидность FORTS «сдуло»: ближние серии (1–2 мес) живы, дальние пусты [00:42:37].
- На РФ — специализироваться на одном инструменте: фьючерс/опционы на **индекс МосБиржи (MIX)**; ликвидность перетекла туда из РТС [00:43:41].
- Рубль-доллар будет терять ликвидность (уход в юани/рупии) [00:44:43].
