# Крипто-платформа — паспорт проекта

**Слаг:** `crypto`

**Репозиторий продукта:** `~/Projects/crypto-platform` (создаётся задачей Т-0)

## Что это

Многоуровневая платформа анализа, тестирования и отбора торговых
стратегий на крипторынке. Первый боевой модуль — система распределения
прибыли (CPDS): дисциплинированная фиксация чистой прибыли малыми
траншами и распределение по корзинам. Не торговый бот и не предсказатель
рынка. Принцип: исследовать можно всё, исполнять — только то, что прошло
фильтр (бэктест → бумага → ручное одобрение Owner'а).

## Красные линии (короткая выжимка, полные — в мандате §3)

- Модель НИКОГДА не касается биржевых ключей и не исполняет сделки.
- Живых денег на этапе «данные и стенд» НЕТ нигде. Автомат — только
  на бумажном полигоне.
- Лимиты — в коде; меняются только записанным решением Owner'а.
- Налоги/юрисдикция/банковские шлюзы — вне системы (слово Owner'а 12.06).

## Контракт федерации (§3)

- Паспорт — этот файл.

## Конвейер работ

Admin проектирует и режет ТЗ → Worker (Опус) собирает → независимый
Reviewer ломает → приёмка Admin → этапные переходы — словом Owner'а.
Порядок задач этапа: Т-0 → Т-1 → (Т-2 ∥ Т-3) → Т-4 → (Т-5 ∥ Т-6) → Т-7
(архитектура §13).
