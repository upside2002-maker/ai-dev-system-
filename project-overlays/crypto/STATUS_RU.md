  арифметика на целых без дробных; главный предохранитель цел —
  без он-чейн данных ядро только наблюдает.

Контур данных + ядро решений работают. Ядро предлагает, деньгами не
распоряжается (исполнение и журналы — вне ядра, впереди).

## Две фразы для голоса

По крипте мозг системы собран целиком: он по рыночному снимку решает,
когда отщипнуть прибыль и куда её разложить, но сам денег не трогает и
без проверенных данных только наблюдает. Дальше — бумажный полигон, где
стратегии торгуют в долг понарошку; живых денег нет.
