## Две фразы для голоса

По крипте собран весь стенд: система берёт рыночный снимок, прогоняет
через мозг-ядро, торгует понарошку на бумаге, честно проверяет идеи на
истории и докладывает результат словами. Денег нигде нет, всё на бумаге;
дальше — гонять стратегии и копить статистику до твоего решения.

## ВЕХА: ЭТАП «ДАННЫЕ И СТЕНД» СОБРАН ЦЕЛИКОМ (13.06)

Вся инфраструктура стенда в master (merge fc19979). Цепочка целиком:
данные (Т-1) → он-чейн критичная семья (Т-2) → ядро решений из 4 кусков
(Т-3a/b/c/d) → виртуальный исполнитель (Т-4a) → прогон по истории
с эталонами (Т-4b) → бэктест-движок walk-forward/point-in-time (Т-5) →
отчёты + доклад секретарю «что по крипте» (Т-6).

Система умеет: собрать снимок дня, прогнать через ядро, исполнить на
бумаге, прогнать по истории, честно бэктестить гипотезы (без заглядывания
в будущее) и доложить результат человеческим языком. Денег нет нигде,
всё на бумаге; путь к живым деньгам — только решением Owner'а после
статистики (мандат §5).

Команды (для исполнителя): `cli collect/snapshot/smoke/run-paper/backtest/
status/report`. 276 тестов, кросс-хэш Python==Haskell цел.
