## Сейчас

**Приняты: Т-0 (каркас), Т-1 (данные+снимок), Т-3a/b/c/d (всё ядро).**
- Т-0 — с третьего захода (две дыры канонизации закрыты).
- Т-1 — ревью поймало утечку ключа источника золота в лог + два блокера,
  закрыты; операционный долг в `KNOWN_LIMITATIONS.md`.
- **Ядро решений CPDS (Т-3, четыре куска) ГОТОВО:** Т-3a сигнал
  (классификация снимка), Т-3b деньги (чистая прибыль + распределение),
  Т-3c блокировки перезахода (антиФОМО), Т-3d сборка (модуль решений +
  риск-ворота). Каждый кусок — высший риск, строгий режим, многолинзовое
  ревью + замок tier A (Reviewer-HANDOFF). Денежные находки = 0; вся
  арифметика на целых без дробных; главный предохранитель цел —
  без он-чейн данных ядро только наблюдает.

Контур данных + ядро решений работают. Ядро предлагает, деньгами не
распоряжается (исполнение и журналы — вне ядра, впереди).

## Две фразы для голоса

По крипте мозг системы собран целиком: он по рыночному снимку решает,
когда отщипнуть прибыль и куда её разложить, но сам денег не трогает и
без проверенных данных только наблюдает. Дальше — бумажный полигон, где
стратегии торгуют в долг понарошку; живых денег нет.

критичная семья оценки).** Две колеи шли параллельно (Т-2 — в другой

Заметка: контрольное ревью Т-2 один раз дало ЛОЖНЫЙ отказ (ревьюер на

с проверкой коммита. Красный итог ≠ правда.

## ВЕХА: бумажный полигон собран целиком (13.06)

Принят Т-4b (прогон по истории + эталоны + отчёт). Цепочка замкнута:
данные (Т-1) → он-чейн критичная семья (Т-2) → ядро решений (Т-3) →

Честная оговорка (Я-4, в KNOWN): сравнение «стратегия против эталонов»
имеет структурную асимметрию — стратегия держит буфер вне рынка, эталоны
целиком в рынке. На докладе статистики Owner'у асимметрию называть явно;
дисклеймер дозреет в отчётном слое Т-6.
