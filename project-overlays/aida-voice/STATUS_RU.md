## Сейчас

**Базовый слой закрыт живым тестом №4 (2026-06-11).** Наша часть — уши,
ключ, стоп, окно диалога — впервые отработала чисто: страховка зависшего
датчика озвучки спасала трижды (болезнь реально повторяется — лечится
сама), цепочка из пяти вопросов прошла живым диалогом без повторной
«Аиды», ложных пробуждений ноль (вся диктовка Owner'а в другое окно
отсеяна), каждый отказ виден в журнале.

Мёртвая зона на границе конца ответа (поправка Owner'а после теста №4)
**закрыта офлайн этапами 1.4 + 1.5, два приёма проверяющим подряд**:
- найдена молчаливая ветка — фраза, начатая в последние секунды её
  ответа, копилась в буфере «стопа» и при смене состояния выбрасывалась
  без следа; больше ни одна фраза не исчезает молча;
- конец её речи теперь вычисляется точно (система знает, сколько звука
  проиграть) — окно разговора открывается через долю секунды после
  последнего слова, а не через страховочные 3 секунды;
- фраза на границе уходит продолжением разговора в модель — «а почему
  не знаешь» через 2–3 секунды получит ответ;
- её собственное эхо отфильтровано: живая речь продолжается после конца
  её звука, эхо обрывается вместе с ним; после «стоп» окно не
  открывается (перепроверено отдельно).
127/127 проверок, корпуса без просадок. Известная тонкость: фраза,
начатая ПОД самый конец её ответа, может потерять первый кусок — в
протоколе теста №5 проверяем прицельно.

Слух самой модели (числа) и свежие данные — без изменений, лечатся в
Phase 2 (текстовая страховка команд + веб-поиск).

Живой тест №5 (первый ход) вскрыл и закрыл ещё два бага (этапы 1.6 +
1.6.1, два приёма проверяющим):
- **обрыв ответа на полуслове** — наша же гонка: на один вопрос успевало
  уйти два запроса ответа, второй убивал первый. Теперь замок: один ход —
  ровно один ответ (заклинивание лечится само);
- **переспрос, съеденный ложным «стопом»** — длинная фраза поверх её
  речи больше не может быть «стопом». «Стоп» — короткое слово, или
  стоп-слово первым в длинной тираде («хватит уже, замолчи» — работает).
  А длинная фраза с «Аидой» первым словом — теперь **перебив**: она
  замолкает и отвечает на новый вопрос.

## Ждёт твоего решения

Живой тест №6 по слову «запускай»: (1) «Аида, кто президент» — ответ
дозвучивает до конца, без обрывов; (2) перебей её посреди ответа:
«Аида, <новый вопрос>» — должна замолчать и ответить на новый;
(3) «стоп» коротко и «хватит уже, замолчи» длинно — оба затыкают;
(4) переспрос через 2 секунды после конца ответа — подхватывает.
