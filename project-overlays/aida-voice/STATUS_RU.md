**Базовый слой закрыт живым тестом №4 (2026-06-11).** Наша часть — уши,
ключ, стоп, окно диалога — впервые отработала чисто: страховка зависшего

сама), цепочка из пяти вопросов прошла живым диалогом без повторной
«Аиды», ложных пробуждений ноль (вся диктовка Owner'а в другое окно
отсеяна), каждый отказ виден в журнале.

Остаточное после теста №4 (с поправкой Owner'а — первичная трактовка
Admin'а про «окно 8 с» была неверна):
- **мёртвая зона 2–4 секунды сразу после конца её ответа** — признак
  «она ещё говорит» снимался страховкой с опозданием, и фраза, сказанная
  в эту зону, пропадала, причём без следа в журнале (щель видимости).
  **Этап 1.4 в работе**: конец озвучки будет считаться точно (по объёму
  записанного звука), граничные фразы перестанут теряться, все ветки
  станут видимыми. Длина окна (8 с) ни при чём — не трогаем.
- модель внутри ослышивается в числах своим слухом («умножь на 54» →
  «на 4») и даже арифметику путает — это не ворота; в командах хозяйства
  числа продублируем текстом местной расшифровки (Phase 2);
- свежих данных нет — закроет веб-поиск (Phase 2).
