# Статус — Astro

**На 6 июня 2026.** Инструмент для астролога Марины: считает соляр-консультацию и выдаёт готовый PDF + веб-интерфейс для ведения клиентов. Ядро расчётов — отдельный надёжный движок, всё остальное (база клиентов, PDF, сайт) — вокруг него.

История событий и техдетали — в `OPERATING/journal/`. Дорожная карта и решения — в `ARCHITECTURE/`. Новые задачи прилетают в `MAILBOX/to-tl-astro.md`.

## Что уже работает

- **Соляр-консультация в PDF.** Натальная карта, распределение планет по домам соляра, темы года, прогноз по транзитам, итоги консультации, раздел «полезные люди». Текст человеческий, без воды.
- **Ближайшие возвраты планет** (новое). Для клиента видно: когда у него ближайший возврат каждой из 10 планет. И в веб-карточке (на сегодня), и в PDF. Медленные планеты (Сатурн, Уран и выше) показываются серией проходов, внешние — с пометкой «раз в жизнь» / «вне продолжительности жизни».
- **Удобство ввода.** Город → автоматически подставляются координаты и часовой пояс. У соляра можно задать «место встречи» (где человек реально находится в день рождения).
- **Профориентация по методу Дарагана** (новое). По натальной карте — топ-3 профессиональных сочетания «планета · дом · знак» с черновым толкованием (через 4 «денежных» дома). И в PDF, и в веб-карточке. Движок проверен против примера из книги Дарагана. Текст толкования лежит в редактируемых правилах — Марина правит сама; помечен «черновое, не финальный голос».
- **База клиентов** в веб-интерфейсе: список → карточка → создать консультацию → скачать PDF.

## Сейчас

Крупных работ в процессе нет. Последнее — закрыли эпик профориентации целиком (движок → толкование → показ в PDF и UI), оба этапа ядра прошли независимую проверку.

## В очереди

1. **Допилить профориентацию** (мелочи, не горят): черновик берёт «способ дохода» по приоритетному дому — для точности по методу стоит учитывать дом, где планета *стоит* («успех через самозанятость»); Марина может поправить это в правилах сама. Плюс косметика (задвоенный заголовок «черновое»). Полный голос Марины в толковании — отдельный трек.
2. **Мелкий баг возвратов** — если спросить ровно в середине текущего возврата Сатурна/Юпитера, путаются подписи проходов (даты при этом верные). Низкий приоритет.
3. **Развитие методики возвратов** — самоаспекты планет, проработка домов через куспиды, глубокие толкования, Чёрная Луна (Лилит). Будущие волны. По Лилит нужно одно решение от Марины (расчёт «средняя» или «истинная»).

## Известные долги (не горят, на заметку)

- Автотесты гоняются вручную на машине владельца — нет автоматической проверки при каждом изменении.
- Резервная копия данных лежит на том же диске — нет независимой внешней копии.
- В тестовых файлах есть реальные данные рождения клиентов без обезличивания — стоит анонимизировать.

## Где смотреть вживую

Поднять всё одной командой: `./run-local.sh` (из папки продукта). Потом веб-интерфейс на `localhost:3000` → клиент → разделы «Ближайшие возвраты» и «Профориентация». Готовый PDF — скачать из карточки консультации.
