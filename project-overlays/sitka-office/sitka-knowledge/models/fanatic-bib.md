# Fanatic Bib

## Короткое описание

Fanatic Bib — тёплый bib (комбинезон с нагрудником) для late-season whitetail hunting в frozen conditions. Наружу — Berber fleece для тихости и концеалмента, WINDSTOPPER by GORE-TEX LABS линер для 100% windproof, PrimaLoft Silver insulation с compression-resistant зоной в седалище. Нижние ноги — 3-layer GORE-TEX с DWR. Mapped burr-resistant textile на hem и seat. Вес 53.1 oz (≈1505 g).
