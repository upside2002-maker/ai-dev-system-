# Stratus Pant

## Короткое описание

Тёплые ветрозащитные штаны на **WINDSTOPPER by GORE-TEX Labs** с тихой микрофлисовой оболочкой и внутренним тёплым флисом. Трёхслойная конструкция для долгого сидения в холод. Комплект к Stratus Jacket.

## Для каких условий

Стрельба из лука в холодный гон, tree stand, засидка — любая статичная охота где важна тишина и защита от ветра. Внутренний warm mesh fleece даёт комфорт при сидении несколько часов. Покупатели отмечают: «комфортны в сидячем положении» — это проектная цель, не случайность.

**Типичный сценарий**: ноябрь, засада на гона, сидишь на платформе 4-6 часов, температура от 0 до -10°C, ветер. Штаны не продувают и не шумят при малейшем движении.

## Отличия от похожих моделей

- **Timberline Pant** — softshell для активной горной охоты. Stratus — для сидения в холод. Противоположные профили.
- **Dew Point Pant** — rain shell (только дождь). Stratus — ветер + тепло.

## Характерные вопросы клиентов

*Секция будет наполнена на основе реальных переписок из Avito/Telegram. Пока — гипотетические вопросы:*

- *«Для ходовой охоты подойдут?»* → Нет. Трёхслойная конструкция тяжёлая и плотная. Для ходовой — Timberline.
- *«Греют в минус?»* → Да, до -10°C комфортно. В более глубокий мороз — с baselayer под ними.
- *«Шумят при движении?»* → Minimum. Микрофлисовая оболочка специально для тишины.
- *«Подойдут для лучника?»* → Да, это одно из целевых применений — тихая стрельба из лука.
