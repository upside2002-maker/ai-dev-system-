# Карта потока капитала — концепт-эскиз (видение Owner'а 14.06)

**Дата:** 2026-06-14
**Автор:** Admin (архитектор), по развёрнутой идее Owner'а
**Статус:** концепт-видение; вход для факторов этапа Т-7 (комплексный анализ)
**Связано:** `charters/crypto-bot.md` §4-бис (платформа), исследование §8.3
(сигналы не доказаны — карантин), KNOWN Д-1/Я-* , Т-7 (проверка факторов)

## Идея (словами Owner'а)

Рынок ходит не «от ликвидности к ликвидности» сам по себе — за движением
стоит, набирает крупный капитал или разгружается. Когда у плечевых лонгов
сбривает миллиарды, форсированный объём кто-то поглощает — поглотить каскад
в моменте может только глубокий капитал. Систематически отслеживая эти
СЛЕДЫ ПЕРЕДАЧИ, можно оценивать интерес крупного: «какой смысл тащить цену
вверх, если они уже разгрузились». И эту картину читать на трёх горизонтах:
спекулятивном (дни), среднесрочном (недели), стратегическом (фазы цикла).

По сути — модель накопления/распределения («умные деньги», фазы Уайкоффа),
но строим её НЕ по нарративу, а по измеримым следам.

## Что ИЗМЕРИМО (следы — собираем/собираем)

Карта сводит уже имеющиеся/накапливаемые сигналы в единую картину
«накопление ↔ распределение»:
- **Ликвидации** (лонги/шорты, по биржам) — где вынесли плечо, какую
  сторону поглотили. Накопление начато (Т-7b).
- **Потоки бирж** (он-чейн): отток на холодные кошельки = унесли (набор);
  приток на биржи = готовят продажу. Есть (Т-2/снимок, сырьё).
- **Открытый интерес + ставка финансирования** — растёт плечо или гасится,
  кто перегружен. Есть (Т-1, семья «плечо рынка»).
- (Позже) стейблкоины на биржах — «топливо» для покупок.

Связка сильнее частей: каскад ликвидаций + одновременный отток монет на
холод = подтверждение, что крупный реально забрал и унёс, а не маркет-мейкер
прокрутил.

## Где ГРАНИЦА (домысел — держать честно)

- Видим СЛЕДЫ, не намерения. «Крупный разгрузился» — вероятностный вывод
  из отпечатков, НЕ факт. Не оракул.
- Источники неполны (биржи занижают ликвидации; потоки бирж — атрибуция
  кошельков шумная). Карта — про ОТНОСИТЕЛЬНУЮ динамику и дисбаланс, не про
  точные миллиарды.
- Красная линия мандата §7: НЕ строить «ИИ, предсказывающий рынок».
  Карта — измеримое состояние + ГИПОТЕЗА о его предсказательной силе,
  проверяемая на стенде, а не вера.

## Двойное назначение (важно)

1. **Карта-состояние (описательная)** — можно строить по мере накопления и
   использовать для ПОНИМАНИЯ и ДОКЛАДА: «крупный сейчас набирает или
   разгружается, уверенность такая-то, по таким следам». Ценно само по
   себе, до сделок. Идёт в отчётный слой (Т-6) как контекст, НЕ как команда.
2. **Торговый сигнал из карты** — ТОЛЬКО через трубу проверки (Т-5/Т-7):
   формализовать «исчерпание интереса крупного → разворот/стоп топлива»,
   прогнать walk-forward без заглядывания в будущее, в сделки — лишь если
   предсказательная сила устоит после издержек. Иначе — нарратив, который
   ложится на прошлое и врёт на будущем (урок исследования §2.1).

## Три горизонта (один набор данных, разные окна)

- Спекулятивный: дни (всплески ликвидаций, резкие потоки).
- Среднесрочный: недели (накопление/распределение тренда потоков).
- Стратегический: фазы цикла (месяцы; стыкуется с он-чейн оценкой —
  семья valuation, когда наберётся история, KNOWN Д-1).

## Место в комплексе

Это ВЕРХНИЙ составной фактор над сырьём (ликвидации + потоки + плечо).
Ложится в семью «плечо рынка»/новую семью «поток капитала» — после того
как проверка покажет силу. До тех пор — описательная карта для доклада.
Путь стандартный: накопить → формализовать → бэктест → доказать →
ввести в сигнал с прозрачным весом. Никакого чёрного ящика.

## Ближайшие шаги

1. Т-7b (идёт): накопление ликвидаций + разведка источников + стык с потоками.
2. После накопления: формализовать «карту потока» как составной фактор +
   прогнать через стенд (предсказывает ли дисбаланс/исчерпание разворот).
3. Параллельно: описательная карта-состояние в доклад (Т-6) — «набирают/
   разгружаются» как контекст, честно с уровнем уверенности.
